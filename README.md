# Карта кругосветной океанологической экспедиции

Интерактивная карта маршрута яхтенной экспедиции с батиметрией GEBCO и морскими зонами (Marine Regions / UNCLOS).

## Маршрут

Марокко → Кабо-Верде → Антильские о-ва → Панама → Панамский канал → Галапагосы → Таити → Фиджи → Торресов пролив → Таиланд → Шри-Ланка → Оман → Джибути → Египет → Суэцкий канал → Турция

## Возможности

- **Географическая карта** — рельеф дна GEBCO, маршрут, контрольные точки
- **Морские зоны** — территориальное море (12 ММ), ИЭЗ (12–200 ММ), открытое море (>200 ММ)
- **Линейная развёртка** — прокрутка маршрута слева направо как одной непрерывной линии с профилем батиметрии и цветовой кодировкой зон
- **Редактирование маршрута** — перетаскивание, добавление и удаление контрольных точек (сохраняется в localStorage)

## Запуск

```bash
npm install
npm run dev
```

Откройте http://localhost:5173

## Публикация на GitHub

1. Создайте public-репозиторий `expedition-route-map` на https://github.com/new
2. В **Settings → Pages → Build and deployment** выберите **GitHub Actions**
3. Выполните:

```bash
git remote add origin https://github.com/frankhorr/expedition-route-map.git
git push -u origin main
```

Сайт соберётся автоматически через GitHub Actions.

## Источники данных

| Данные | Источник |
|--------|----------|
| Батиметрия | [GEBCO WMS](https://www.gebco.net/data-products/gebco-web-services/web-map-service) |
| ИЭЗ, территориальные воды, открытое море | [Marine Regions / VLIZ](https://www.marineregions.org/) |

## Примечание

Классификация морских зон на развёртке выполняется через WFS-запросы к Marine Regions и может занять 1–2 минуты при первой загрузке. Для работы нужен доступ в интернет.
